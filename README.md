# The-InsecuriTea-Project-Breaking-and-Hardening-a-Web-Application
InsecuriTea is a vulnerable Flask web app demonstrating OWASP Top 10 flaws like SQLi, XSS, and IDOR. Users can register and manage tea preferences, while vulnerabilities are intentionally exploitable. A secure version, SecuriTea, fixes these using parameterized queries, output encoding, and proper authorization.
